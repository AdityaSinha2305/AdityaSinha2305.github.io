PROJECTS

BrainAI

The Brain Tumor Detection Project employs state-of-the-art deep learning and medical imaging techniques to accurately identify and categorize brain tumors from MRI scans. This innovative system plays a crucial role in early diagnosis and treatment planning, potentially saving lives.

Student Tracker

The Student Tracker Project is a focused system designed to manage crucial course-related details of students. It enables easy tracking of enrollment, course progress, and grades. This project simplifies administrative tasks, ensuring efficient management of student data within the academic context.

GetSetGo Booking

The Bus Ticket Reservation is a user-friendly system allows passengers to effortlessly book bus tickets online, view schedules, and manage reservations. It also provides administrators with tools for efficient ticket management and real-time monitoring, ensuring a seamless and organized travel experience for users.

Captcha for visually impaired

This project is a vital accessibility initiative. It utilizes innovative audio and tactile methods to provide an inclusive solution for visually impaired individuals encountering Captcha challenges online. This project aims to bridge the digital accessibility gap, ensuring that everyone can participate equally in online interactions.
